# Системные требования

## Операционная система:
- Windows 10 или выше

## Платформа:
- .NET Framework 4.7+
- WPF (C#)

## База данных:
- MS SQL Server 2016+

## Требования к памяти:
- Минимум 2 ГБ ОЗУ
- 1 ГБ свободного пространства на диске

## Сетевые требования:
- Подключение к локальной или удаленной БД (при необходимости)
