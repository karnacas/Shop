# ToolsShop

## Описание
GUI-приложение для магазина инструментов на WPF, Entity Framework и MS SQL Server.

## Функции:
- Авторизация.
- CRUD операции.
- Анализ продаж.
- Инициализация случайных данных (Файл → Инициализировать).

## Технологии
- C# + WPF
- Entity Framework 6.4
- MS SQL Server
